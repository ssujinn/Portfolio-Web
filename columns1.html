Python Interpreter

Python

Python은 고급 프로그래밍 언어로, 플랫폼 독립적이며 인터프리터식, 객체지향적, 동적 타이핑(dynamically typed) 대화형 언어이다. python의 string 처리 함수 중 몇 가지를 설계한다.

C언어를 사용하여 프로그램을 구성하되 입력에 대한 동작은 Python과 동일하도록 한다.
설계할 함수는 다음과 같다.

  • capitalize()
  • swapcase()
  • title()
  • istitle()
  • lower()
  • upper()
  • islower()
  • isupper()
  • isalpha()
  • isalnum()
  • isspace()
  • lstrip()
  • rstrip()
